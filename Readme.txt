### Prerequisites

- [Node.js](https://nodejs.org/) (if you wish to develop frontend assets)
- [XAMPP/LAMP/WAMP](https://www.apachefriends.org/index.html) or any web server with PHP support (for running PHP components)

### Installation

1. **Clone the Repository**
    ```bash
    git clone https://github.com/manojmahadevappa/Yashus-Kitchen.git
    cd Yashus-Kitchen
    ```

2. **Serve Your Project**

   - For static preview: Simply open `index.html` in your browser.
   - For PHP features: Place the files in your web server's root folder (e.g., `htdocs/` for XAMPP) and open `http://localhost/Yashus-Kitchen/` in your browser.
